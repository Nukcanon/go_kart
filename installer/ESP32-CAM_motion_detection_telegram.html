ESP32-CAM 모션감지 텔레그램 전송
OneClick uploader

  1. ESP32-CAM을 flashing 모드로 설정하여 컴퓨터와 연결합니다.
    flashing 모드 진입 방법
    아래 회로도 사용 시: IO0와 GND 연결
    전용 programmer board 사용 시: boot(혹은 IO0)버튼 누른상태 -> reset버튼눌럿다 떼기 -> boot(혹은 IO0)버튼 떼기
  2. 다른 ESP32-CAM을 사용하는 프로그램(Arduino IDE 등)을 모두 끕니다.
    특히 Arduino IDE의 시리얼 모니터가 켜져있으면 업로드가 불가능합니다.
  3. 이 페이지 가장 아래의 Connect 버튼을 누른 후, 팝업창에서 ESP32-CAM의 포트를 선택하여 연결합니다.
  4. 팝업창에서 INSTALL을 선택하면 프로그램 업로드가 시작됩니다.
  5. 업로드가 완료되면 flashing 모드를 해제한 후, ESP32-CAM을 reset 합니다.
    flashing 모드를 해제 방법
    아래 회로도 사용 시: IO0와 GND 연결 해제 후 reset버튼 누르기
    전용 programmer board 사용 시: reset버튼 누르기
  6. 아래의 Connect 버튼을 다시 누른 후 LOGS & CONSOLE에 진입하여, 오른쪽 아래의 RESET DEVICE 버튼을 누르면, 해당 ESP32의 이름을 확인할 수 있습니다.
    디바이스 이름은 "ESP32-1234"와 같이 4자리 숫자로 이루어져 있습니다.
  7. 스마트폰이나 컴퓨터의 Wi-Fi로 확인한 이름을 가진 ESP32와 연결 한 후, 인터넷 브라우저로 http://192.168.4.1 에 접속하여 WiFiManeger를 설정합니다.
